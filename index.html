# ชุดเกมส์ทัศนศิลป์ออนไลน์ ไฟล์ HTML ทั้งสองนี้สามารถเผยแพร่บน GitHub Pages เพื่อให้ผู้เรียนเข้าเล่นได้ผ่านเว็บเบราว์เซอร์ ## เนื้อหา 1. `art_elements.html` - เกมส์เรียนรู้และทดสอบเรื่องทัศนธาตุ องค์ประกอบศิลป์ และ Perspective 2. `color_theory.html` - เกมส์เรียนรู้และทดสอบเรื่องทฤษฎีสี ## วิธีใช้งานบน GitHub Pages 1. สร้าง Repository ใหม่ใน GitHub (Public) 2. อัปโหลดไฟล์ HTML เหล่านี้เข้า Repository 3. ไปที่ **Settings > Pages** 4. เลือก Branch เป็น `main` และ Folder เป็น `/root` (หรือ `/docs` ถ้าจัดเก็บในโฟลเดอร์นั้น) 5. ระบบจะสร้าง URL เช่น `https://.github.io//` 6. ส่งลิงก์ให้ผู้เรียนเพื่อเข้าเล่นเกมส์ ## การนำไปฝังในเว็บไซต์อื่น - สามารถนำ URL ที่ได้ไปฝังใน Canva Website หรือ Google Sites ได้ โดยใช้ปุ่มหรือ iframe
